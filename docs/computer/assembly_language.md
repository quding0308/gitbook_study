## 汇编语言

Assembly Language 简称 asm

对于人类来说，二进制程序是不可读的，根本看不出来机器干了什么。为了解决可读性的问题，以及偶尔的编辑需求，就诞生了汇编语言。

汇编语言是二进制指令的文本形式，与指令是一一对应的关系。

每一种 CPU 的机器指令都是不一样的，因此对应的汇编语言也不一样。
